API Guide

Hướng dẫn sử dụng Masothue API Proxy

Đây là trang tài liệu API. Giao diện / chỉ dành cho test nhanh. Tất cả client thật nên gọi API JSON ở /api/*.

Base URL:

API trả dữ liệu gọn từ masothue.com và chuẩn hóa trạng thái phản hồi để client dễ xử lý.

Endpoint

GET /api/lookup

Tra cứu linh hoạt theo từ khóa.

Query

  • q hoặc query: từ khóa cần tra cứu
  • type: kiểu tra cứu
curl 'http://localhost:3000/api/lookup?q=1102168726&type=enterpriseTax'

GET /api/company/:taxCode

Tra cứu nhanh theo mã số thuế doanh nghiệp.

Ví dụ

curl 'http://localhost:3000/api/company/1102168726'

Kiểu tra cứu

Doanh nghiệp / công ty

  • enterpriseTax: mã số thuế doanh nghiệp
  • enterpriseName: tên công ty / doanh nghiệp
  • legalName: giám đốc / người đại diện

Cá nhân

  • personalTax: mã số thuế cá nhân
  • identity: CMND / CCCD
  • auto: tự động suy luận kiểu phù hợp

Trạng thái phản hồi

200 OK

Trả về khi tìm thấy dữ liệu đủ mạnh.

404 NOT_FOUND

Trả về khi không tìm thấy kết quả phù hợp.

  "ok": false,
  "code": "NOT_FOUND",

Ghi chú

  • Giao diện test ở / chỉ để kiểm tra nhanh.
  • Client thật nên đọc ok, code, reason và HTTP status.
  • Khi cần debug sâu, dùng /api/debug-lookup.
